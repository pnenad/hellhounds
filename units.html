Unit details:

Each unit has an ability, and each ability has a primary and secondary effect. An ability also has a cost in resources. You might see that some abilities have a cost in the "Any" resource, this means that it's a wildcard cost, and will accept any resource as payment.

Here is a list of our current units.

NOTE: When creating new units its important to consider that each unit must be able to deal some amount of damage, and a unit should not be able to give more or equal healing when compared to damage. This could result in no-win scenarios.

For instance consider if vampires healed for the same amount of (or more than) damage they dealt. In a game state where each player only had a vampire left they could then be biting each other indefinitely provided both players had resources to do so.
