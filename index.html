Welcome to Hellhounds Semester Project.

This is the place where you'll be able to follow the project status, progress, ideas etc. Feel free to download and compile our source code, give it a try and make sure to visit our GitHub pages and give us a comment.

The project is to create a two-player, turn based game, with some similarity to card games like Hearthstone or Magic: the gathering. We want the game to be playable on both PC and android mobile devices. Our working title is BattleFree, though we plan to launch the game as OpenWarfare.

To reach our goal we need the following:

A set of game rules, and a game engine able to enforce those rules.

Network functionality, at least between two clients through a server. Also a stable server that can handle client game-requests.

Graphic user interfaces for both the PC client and the mobile client.

To see our progress, you can download, compile and run our repository using git: 

$git clone https://github.com/pnenad/hellhounds.git
$make
$./run
